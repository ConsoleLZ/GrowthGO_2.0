# 欢迎来到我的博客

欢迎来到我的个人博客！这里将分享我的技术心得、项目经验和学习笔记。

## 博客功能特色

- **Markdown 支持**：所有文章都使用 Markdown 格式编写
- **标签分类**：每篇文章都可以添加多个标签，方便分类浏览
- **搜索功能**：支持按标题、描述和标签搜索文章
- **响应式设计**：适配各种设备屏幕尺寸
- **深色模式**：支持深色和浅色主题切换

## 如何使用

1. 在博客页面浏览所有文章
2. 使用搜索框快速查找感兴趣的内容
3. 点击文章标题查看完整内容
4. 通过标签筛选相关文章

## 技术栈

这个博客功能基于以下技术构建：

- **Next.js 14**：React 框架，支持服务端渲染
- **TypeScript**：提供类型安全的开发体验
- **Tailwind CSS**：实用的 CSS 框架
- **gray-matter**：Markdown 文件解析
- **remark**：Markdown 到 HTML 的转换
